## DuplicateFinder

Консольная утилита на .NET 9 для поиска дубликатов файлов по содержимому (SHA-256).

## Возможности

- Рекурсивный обход папки
- Подсчет SHA-256 для каждого файла
- Группировка файлов по хешу
- Вывод только реальных дубликатов (группы от 2 файлов)
- Итоговая статистика:
  - количество групп
  - количество файлов в группах
  - количество лишних копий

## Стек

- C# / .NET 9
- xUnit (тесты)

## Структура проекта

- `DuplicateFinder.Core` - бизнес-логика (сканирование, хеширование, поиск групп)
- `DuplicateFinder.Cli` - консольный интерфейс
- `DuplicateFinder.Core.Tests` - автотесты

## Как запустить

### 1) Сборка

```bash
dotnet build
```

### 2) Тесты

```bash
dotnet test
```

### 3) Запуск CLI

```bash
dotnet run --project DuplicateFinder.Cli -- "C:\Path\To\Folder"
```

## Пример вывода

```text
Group 1:
hash - 2CF24DBA5FB0A...
    C:\Test\folder1\file1.txt
    C:\Test\folder2\file2.txt

Количество групп: 1
Количество файлов в группах: 2
Количество лишних копий: 1
```

## Обработка ошибок

- Если путь не указан - выводится Usage и код завершения `1`
- Если путь некорректный или произошла ошибка чтения - выводится сообщение об ошибке

## Покрытие тестами

Проект покрыт unit-тестами на ключевые сценарии:

- поиск дубликатов при одинаковом контенте
- отсутствие дубликатов при уникальных файлах
- пустая директория
- директория с одним файлом
- смешанные наборы файлов
- тесты на `FileScanner`, `FileHasher` и `DuplicateFinderService`
