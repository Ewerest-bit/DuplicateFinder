\# DuplicateFinder

Консольная утилита на .NET 9 для поиска дубликатов файлов по содержимому (SHA-256).

\## Возможности

\- Рекурсивный обход папки

\- Подсчет SHA-256 для каждого файла

\- Группировка файлов по хешу

\- Вывод только реальных дубликатов (группы от 2 файлов)

\- Итоговая статистика:

&#x20; - количество групп

&#x20; - количество файлов в группах

&#x20; - количество лишних копий

\## Стек

\- C# / .NET 9

\- xUnit (тесты)

\## Структура проекта

\- `DuplicateFinder.Core` — бизнес-логика (сканирование, хеширование, поиск групп)

\- `DuplicateFinder.Cli` — консольный интерфейс

\- `DuplicateFinder.Core.Tests` — автотесты

\## Как запустить

\### 1) Сборка

```bash

dotnet build

2\) Тесты

dotnet test

3\) Запуск CLI

dotnet run --project DuplicateFinder.Cli -- "C:\\Path\\To\\Folder"

Пример вывода

Group 1:

hash - 2CF24DBA5FB0A...

&#x20;   C:\\Test\\folder1\\file1.txt

&#x20;   C:\\Test\\folder2\\file2.txt

Количество групп: 1

Количество файлов в группах: 2
